
<h1 align="center"> peditor </h1>
<p align="center">A pythonian text editor</p>

### Requirements
* Linux
* python 3

### Installation
```sh
$ git clone https://github.com/shubham1172/peditor.git
$ sudo ./install.sh
``` 
### Running
```sh
$ peditor [filename]
```
