
<h1 align="center"> peditor </h1>
<div align="center">A pythonian text editor</div>

<center><img src="./docs/images/peditor.png"></center> 

### Requirements
* Linux
* python 3

### Installation
```sh
$ git clone https://github.com/shubham1172/peditor.git
$ cd peditor
$ ./scripts/install.sh
``` 
To remove the build,
```sh
$ ./scripts/clean.sh
```
### Running
```sh
$ peditor [filename]
```
